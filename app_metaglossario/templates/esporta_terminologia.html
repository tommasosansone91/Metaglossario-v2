Esporta terminologia tramite API

Il metaglossario del progetto Gestisco dispone di servizi API in stile REST che permettono ad altre applicazioni o script di accedere alla terminologia. Il servizio permette l'accesso alla terminologia desiderata attraverso un paradigma di URL statico e illustrato di seguito. Eventuali errori nell'utilizzo delle API vengono segnalati attraverso codici di risposta HTTP. Per tutte le richieste e le risposte dall'API viene utilizzato il formato JSON.
